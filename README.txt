This folder contains a ready-to-deploy static website for DES Tech.

How to deploy to GitHub Pages:
1. Create a public repository named: <your-github-username>.github.io
2. Upload all files (drag and drop) to the repository root.
3. Commit and push changes.
4. Visit: https://<your-github-username>.github.io

Optional:
- To use a custom domain, add a CNAME file with your domain name and configure DNS as A/ALIAS records.
